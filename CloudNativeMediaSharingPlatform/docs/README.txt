Cloud Native Media Sharing Platform

Frontend:
- Static HTML + JS for uploading and viewing images
- Use Bootstrap/jQuery (lightweight)

Backend:
- Use Azure Logic Apps for REST API endpoints
- Store files in Azure Blob
- Store metadata in Azure Cosmos DB

Setup:
1. Replace YOUR_IUPS_ENDPOINT and RAI with actual Logic App endpoints
2. Replace BLOB_ACCOUNT with your Blob Storage base URL
3. Deploy frontend to Azure Static Web Apps or any static host
4. Follow Logic App JSON to configure your backend
